# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-19

### Added
- **Import Engine**: Markdown, TXT, GitHub Issues, GitHub PRs importers
- **AI Agents**: Conversation, Task, Sentiment, Entity agents
- **Workspace**: Project AI analysis view with task board and sentiment metrics
- **Dashboard**: Cross-project analytics and insights
- **AI Chat**: Natural language querying of project data
- **Knowledge Graph**: Interactive entity relationship visualization
- **Reports**: Multi-format export (PDF, Markdown, JSON)
- Clean Architecture backend with DDD principles
- Modern uv package management
- Docker deployment configuration

### Features
- Project CRUD with status management
- Communication event extraction and storage
- AI-powered summary, decision, and topic extraction
- Task extraction with priority and assignee detection
- Sentiment analysis with stress indicators
- Entity extraction (people, technologies, APIs, etc.)
- Dark theme SaaS-quality UI
